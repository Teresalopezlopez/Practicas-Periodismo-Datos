Uso de la API del Covid 19 con Pandas¶

Para realizar esta prctica utilizamos Python, un lenguaje de programacin de alto nivel para el desarrollo de software. A diferencia de otros lenguajes como Java, se trata de un lenguaje interpretado, es decir, cuyo cdigo ya es legible y no requiere traduccin.

En este cuaderno Python trabajaremos la representacin de los datos de la API del Covid-19 por pases con la librera Pandas. Como sus siglas indican, Pandas es la librera de Python para el anlis de datos (Python Data Analysis Library) que nos es muy til para la manipulacin y anlisis de datos en el lenguaje de progrmacin de Python. Es una biblioteca de software escrita como extensin de NumPy que ofrece estructuras de datos y funcicones para manipular tablas numricas y series temporales.

La Url con la que vampos a trabajar es la siguiente: https://api.covid19api.com/countries

Antes de poder empezar a trabajar con los datos necesitamos realizar tres pasos previos:

  1. Preparar libreras 1.1 Instalar 1.2 Importar
  2. Definir las variables con las que vamos a trabajar, en este caso la API del Covid-19
  3. Crear del dataframe para que Python muestre los datos

Dentro de la funcin read_json ponemos : para poner lo que queremos que lea, La url con los datos est es JSON: son los datos en este lenguaje.
